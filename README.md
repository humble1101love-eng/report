# 高德 × 千问 打车 Agent 交互流程

这是一个展示高德打车与千问 AI 协同交互流程的可视化页面。

## 功能说明

页面展示了从语音唤醒到支付结算的完整打车流程，包含 5 个主要步骤：

1. **语音唤醒与意图识别** - 自然语言指令输入与意图解析
2. **方案推荐与决策** - 路线规划、车型推荐、偏好匹配
3. **确认下单与派单** - 卡片确认与司机接单
4. **行中服务与协同** - 实时轨迹与行中改派
5. **支付结算与售后** - 支付与发票服务

## 图例说明

- 🔵 **千问侧** - LUI/卡片/记忆相关功能
- 🟠 **高德侧** - 运力/调度/交易相关功能
- 🌈 **千问&高德打车** - 双方协同功能
